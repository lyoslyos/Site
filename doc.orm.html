Active Record
Introduction
Active Record est un ORM simple qui vous permettra de faire toutes vos actions CRUD, mais dispose également d'un query builder. Il s'adapte plutot bien a vos models et aux relations entre les tables. Nous allons ici voir quelques basique pour le CRUD. Pour le reste je vous renvoi vers la documentation complète de php Active Record.
CRUD
Create :

Pour créer un nouveau user : User::create(array('name' => 'Tito', …));
create va attendre un array comprenant le nom du champ puis sa valeur


Read :

Pour lire des données on peut très facilement récupérer un utilisateur par son id
Comme ceci : $user->find(1); // on récupère ici l'utilisateur avec l'id = 1
On peut également récupérer tous les utilisateurs avec : $user->find('all');

Active Record est de plus assez intéligent pour permettre de faire des méthodes personnalisés aux models.
exemple : $user->find_by_name('Michel'); // va directement chercher tous les utilisateurs ou le nom seras Michel


Update :

pour ceci une fois l'instance d'un utilisateur modifié par rapport au donnée enregistré en Base de Donnée.
Il nous suffit de faire : $user->save();
Ce qui sauvegarderas en base de données les nouvelles données;


Delete

Pour delete une ligne en base de donnée. Une fois l'instance hydraté.
Il nous suffit de faire $user->delete();


Comme vous avez pu le voir Active Record est très simple d'utilisation mais réserve tout de même quelques bonnes surprise ! Il vous faut absolument aller voir la documentation de Active Record pour pouvoir l'utiliser dans un projet.
