Twig
C'est très simple !

En reprenant la structure offerte par l'archive, vous retrouvez la mise en place de Twig adèquate. Vous aurez seulement besoin de remplir un tableau avec le contenu à afficher et le nom du template dans vos controllers. 


                        echo $this->twig->render('index.twig', array(
                            'bateau' => 'DrakkarD'
                        ));

index.twig est le template que vous souhaitez afficher.

bateau est le nom utilisé dans le template pour afficher votre string.

DrakkarD est votre string, ce peut aussi être une variable.



Dans votre template, vous afficherez votre contenu ainsi : 


                        Le meilleur framework au monde est : {{ bateau }} !

Et vous obtiendrez :


                        Le meilleur framework au monde est : DrakkarD !


bateau.attribut vous permet d'accéder à un attribut de la classe définie comme bateau.

{{ bateau }} correspond à du contenu dynamique.

{# commentaire viking #} correspond à un commentaire.

{% filter escape %} correspond à une opération, comme if par exemple.



Ceci ne constitue qu'une introduction à Twig et vous êtes invité à vous rendre sur le site officiel ici pour approfondir vos connaissances et à mettre en pratique tout cela avec DrakkarD, le framework qui flotte !
